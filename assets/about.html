OFTW Dashboard – with compliments from the team at Arkimetrix Analytics

Notes

This dashboard summarizes key metrics from the Metric Wishlist. The data pipeline is not currently connected to live source files, but can be scheduled for automated updates as needed. Refresh is not currently enabled.

Targets

A form is available to create or update targets by fiscal year to run different scenarios. Fiscal 2024–25 targets were updated using the provided spreadsheet. Monthly targets reflect prior-year seasonality where possible.

  • KPI cards on top gives the idea of how much achieved of Target for different metrics.
  • Cumulative Money Moved chart shows the journey of MM till FYTD with related labels, also providing view on target run rate based on PY pacing to reach current year's target.
  • Active Pledge ARR (toggle to Current ARR + Remaining to Target)chart shows the run of each chapter type category and its frequency to fulfill the gap to target.

Currently, target value update reflects per session per user only so when refreshed it will change to default. This can be enhanced in the future as needed.

Data Context

Visuals provide both numeric and comparative context to prior year and target values. Views include Monthly and Fiscal YTD perspectives. Future enhancements may support toggles between Monthly, Quarterly, and YTD views.

Date Table

A custom fiscal calendar is used, starting July 1. All aggregations (e.g., YTD) are based on this fiscal calendar.

Filters Applied

The following portfolios have been excluded:

  • One for the World Discretionary Fund
  • One for the World Operating Costs

Duplicate payment records (by ID) were also removed to ensure accuracy.

FX Rate Conversion to USD

Local currencies are converted to USD using monthly FX rates from the Bank of Canada.

  • XXX_to_USD = FXMXXXCAD / FXMUSDCAD
  • CAD_to_USD = 1 / FXMUSDCAD

FX rates are joined on a monthly lookup date, and all pledge and payment values are converted to USD using the corresponding rate. Final values are saved in pledges.parquet and payments.parquet.
